1 /* @brief Standard agglomerative hierarchical clustering methods.
2  * @details
3  * Assume input object number is N, then form a N*N distance matrix to be stored.
4  * Basic procedures are
5  * 1. store distance matrix
6  * 2. min-heap to sort mutual distance
7  * 3. every time merge two nodes with smallest distance and update the min-heap
8  * 4. merge until only one cluster or given cluster number is obtained
9  * Performance and memory usage analysis
10  * 1. Performance
11  * N*N (distance matrix) + 2N*N logN (build min-heap) + 2N*N*log(N*N) (min-heap update)
12  * 2. Memory usage
13  * N*N (distance matrix) + N*N(min-heap)
14  * Would expect this algorithm to be super time-consuming and memory-wasting
15  *
16  * @author Lieyu Shi
17  */
